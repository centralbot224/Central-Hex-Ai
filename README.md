## 🎯 **Présentation**

**CENTRAL-HEX AI** est un assistant intelligent conçu pour aider les utilisateurs à maîtriser l'intelligence artificielle. Il combine :

- 🧠 **Formateur IA** : Apprenez à structurer des prompts puissants
- 📚 **Bibliothécaire** : Accédez à des ressources exclusives
- 💬 **Community Manager** : Interagissez avec une communauté passionnée
- 🎓 **Coach personnel** : Guide pas à pas dans votre apprentissage

> "La meilleure façon de prédire l'avenir est de le créer." - Peter Drucker

## ⚙️ **Installation**

### Prérequis
- Node.js 18+
- npm ou yarn
- Compte OpenAI avec clé API
- MongoDB (local ou Atlas)

### Étapes d'installation

```bash
# 1. Cloner le repository
git clone https://github.com/centralbot224/CENTRAL-HEX-AI.git
cd CENTRAL-HEX-AI

# 2. Installer les dépendances backend
cd backend
npm install

# 3. Configurer les variables d'environnement
cp .env.example .env
# Éditez .env avec vos clés API

# 4. Lancer le serveur
npm run dev
```

## 🔧 **Configuration**

Créez un fichier `.env` à la racine du dossier `backend` :

```env
# Serveur
PORT=5000
NODE_ENV=development

# OpenAI
OPENAI_API_KEY=votre_clé_api_ici

# MongoDB
MONGODB_URI=mongodb://localhost:27017/central-hex

# JWT
JWT_SECRET=votre_secret_jwt_ici
JWT_EXPIRE=30d
```

## 🚀 **Utilisation**

### Démarrer le serveur

```bash
# Mode développement
cd backend
npm run dev

# Mode production
npm start
```

### Tester l'API

```bash
# Test de santé
curl http://localhost:5000/api/health

# Envoyer un message
curl -X POST http://localhost:5000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"Comment structurer un bon prompt ?"}'

# Générer une image
curl -X POST http://localhost:5000/api/image/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt":"Un paysage futuriste avec des robots"}'
```

## 📚 **API Documentation**

### Routes disponibles

| Méthode | Endpoint | Description |
|---------|----------|-------------|
| `GET` | `/api/health` | Vérifier l'état du serveur |
| `POST` | `/api/chat` | Envoyer un message à l'IA |
| `POST` | `/api/chat/optimize` | Optimiser un prompt |
| `GET` | `/api/chat/history/:userId` | Récupérer l'historique |
| `POST` | `/api/image/generate` | Générer une image |

### Exemple de requête `/api/chat`

**Request:**
```json
{
  "message": "Comment améliorer mes prompts ChatGPT ?",
  "userId": "user123"
}
```

**Response:**
```json
{
  "success": true,
  "response": "Voici comment améliorer vos prompts...",
  "timestamp": "2024-01-01T12:00:00Z",
  "tokens": 150
}
```
